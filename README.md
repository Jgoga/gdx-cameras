# GDX-Cameras

Preview: https://youtu.be/cBKBV_m1NdQ

## Install

    allprojects {
	    repositories {
            ...
            maven { url 'https://jitpack.io' }
        }
    }
  
    dependencies {
        compile 'com.github.desenvolvedorindie.gdx-cameras:gdx-cameras:0.0.4'
    }

## Features

- Limit Area
- Follow Player
- Midpoint
- Focal Points (WIP)
- Smooth Position (Interpolation)
- Smooth Zoom (Interpolation)
- Debug
